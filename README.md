## Setup Guide

1. Install the app
2. Set it as assistant via `Apps` > `Default Apps` > `Assist app`
3. Trigger assistant, crop and share


## Features

- Screenshot in 100% JPEG-Quality
- Tight system integration via Assist API
- No permission needed
